# Learn ML From Scratch

A hands-on project for learning machine learning from the ground up — written and explained step by step.

## Levels

| Level | Topic | Folder |
|-------|-------|--------|
| 1 | Build a tiny GPT from scratch | `level1_tiny_gpt/` |
| 2 | Fine-tune with Hugging Face | `level2_finetune/` |
| 3 | Use Claude API | `level3_api/` |

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
